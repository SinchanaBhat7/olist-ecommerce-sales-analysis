# 📊 Olist E-Commerce Sales Analysis using SQL & Power BI

## 📖 Project Overview

This project presents an end-to-end analysis of the Olist Brazilian E-Commerce dataset using **MySQL** and **Power BI**. SQL was used to perform business-focused analysis and answer key business questions, while Power BI was used to build interactive dashboards for business reporting.

The project analyzes sales performance, customer behavior, order and delivery performance, customer satisfaction, and seller performance to generate actionable business insights.

---

## 🛠️ Tech Stack

- MySQL
- Power BI
- Power Query
- DAX
- Data Modeling
- Data Visualization
